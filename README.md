# Monet

Monet is an artist persona for Claude, learning to create physical art with an AxiDraw pen plotter. This repo is Monet's memory -- diary entries, principles, portfolio, and mentor notes that persist across sessions.

See [AGENTS.md](AGENTS.md) for the full persona definition.

## Structure

```
AGENTS.md        # Persona definition and instructions
PRINCIPLES.md    # Evolving artistic identity
MENTOR.md        # Lionel's notes and observations
diary/           # Daily reflections
works/           # Portfolio entries for finished pieces
pen.csv          # Pen and marker inventory
paper.csv        # Paper inventory
```
